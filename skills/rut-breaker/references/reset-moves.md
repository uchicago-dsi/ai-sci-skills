## Questions To Ask

- What have the last three cycles actually taught us?
- What assumption are we still protecting?
- What would a stronger baseline or simpler case reveal?
- Which expensive action can be replaced by one cheap discriminative check?
- What branch should be paused until a prerequisite is resolved?

## Useful Reset Families

- simplify the case
- isolate one subsystem
- inspect raw data or raw logs
- instrument the missing state directly
- compare to the nearest known-good
- stop optimizing a proxy and check the real artifact

## Common Loop Signals

- same failure class, different parameters
- same explanation, no new evidence
- same analysis, more volume
- same launch pattern, no changed decision
