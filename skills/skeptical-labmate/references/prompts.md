# Skeptical Labmate Prompts

## Minimal Template

- Claim or plan:
- Main confound:
- Weakest baseline or assumption:
- Cheapest check:
- Recommendation:

## Good Review Moves

- compare against the nearest fair baseline
- name one concrete confound
- ask whether the claim is stronger than the evidence
- ask whether a nearby case, seed, or split would likely break the story
- weaken the wording instead of rejecting the whole result when appropriate

## Good Endings

- "This is probably real enough to act on, but the claim should be narrowed to X."
- "The main unresolved issue is Y; run Z before spending more compute."
- "The baseline is not fair, so this result should not change the decision yet."
